## 変更履歴

### v1.2.1 (2026-06-10)
- 印刷・結果エクスポート機能（M10）:
  - 🖨 **印刷ボタン**: 成績表をPDF/印刷形式で出力可能
  - 📋 **コピーボタン**: 結果テキスト（スコア・ランク・正答率・難易度・日時）をクリップボードにコピー
  - 📥 **履歴CSVダウンロード**: プレイ履歴をCSV形式でダウンロード（番号/スコア/難易度/日付）

### v1.2.0 (2026-06-10)
- グラフ・スコア可視化（M4）: Chart.jsで直近10回のスコア推移を折れ線グラフで表示
  - 平均スコア・正答率を結果画面に表示
  - プレイ履歴をlocalStorageに記録（難易度別・最大10回分）
- 目標設定・達成度トラッキング（M6）: ゲーム開始前に目標スコアを設定可能
  - 目標達成度メーターを結果画面に表示
  - 目標達成時に特別なメッセージを表示

### v1.1.0 (2026-06-07)
- タイマー機能追加: 難易度別カウントダウン（かんたん12秒/ふつう8秒/むずかしい5秒）、時間切れは不正解処理
- ベストスコア記録: 難易度別にlocalStorageへ保存、更新時バナー表示
- スコア付きSNSシェア: 結果画面からスコア・ランク・難易度を含むツイートが可能
- 段階的難易度: 10問の中で後半ほど色差が小さくなるよう調整
- 使い方セクション: ゲーム専用の正確な説明文に修正

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 色彩感覚クイズゲームは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
